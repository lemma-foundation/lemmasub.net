Setup

Copy, paste, mine.

Set up one OpenAI-compatible provider, one Bittensor hotkey, then let the Lemma CLI guide the proof, commitment, and miner server.

1

Install Lemma.

git clone https://github.com/spacetime-tao/lemma.git
cd lemma
uv sync --extra btcli

2

Paste your provider URL and key.

cat >> .env <<'EOF'
LEMMA_PROVER_BASE_URL=https://your-provider.example/v1
LEMMA_PROVER_API_KEY=replace_me
EOF

3

List models from that URL.

source .env
curl -sS "$LEMMA_PROVER_BASE_URL/models" \
  -H "Authorization: Bearer $LEMMA_PROVER_API_KEY"

4

Paste one model id.

cat >> .env <<'EOF'
LEMMA_PROVER_MODEL=copy_one_model_id_here
EOF

5

Create and register a hotkey.

btcli wallet create --wallet.name lemma --wallet.hotkey miner
btcli subnets show --netuid 467 --network test
btcli subnets register --netuid 467 \
  --wallet.name lemma \
  --wallet.hotkey miner \
  --network test

6

Run setup, then mine.

uv run lemma setup --role miner --wallet lemma --hotkey miner
uv run lemma mine
uv run lemma status

Bounties

Package a bounty proof locally.

uv run lemma mine --bounty <campaign-id> \
  --submission Submission.lean

Validators

Check before start.

bash scripts/prebuild_lean_image.sh
uv run lemma setup --role validator
uv run lemma validate
