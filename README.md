# lemmasub.net

Static GitHub Pages site for Lemma, an AI infrastructure network for verifier-grounded training data.

## Local Preview

```bash
python3 -m http.server 8877 --bind localhost
```

Open `http://localhost:8877/`.

The site has no build step and can also be opened directly from `index.html`.

## Deployment

GitHub Pages serves the repository root. Keep the site static unless there is a strong reason to add a build step.
