What is Lemma?

A Bittensor subnet for rewarding Lean-verified mathematical proofs.

What does a miner do?

Generate a Lean proof out of band, submit it with the CLI, and optionally serve verified proofs to validators.

What does a validator do?

Poll miners, run the pinned Lean checker, and write weights from verification results.

What earns credit?

The exact target theorem proved in Lean. No sorry, no new axioms, no prose score.

What happens on a batch tie?

If multiple miners verify in the same validator batch, the reward is split equally.

Where is the dashboard?

Retired. Current protocol state lives in the CLI and repository docs.
