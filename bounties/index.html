Lemma Bounties

A bounty is a reward for proving one exact Lean statement outside the normal timed dashboard round.

Launch bounty lane

Prove selected Formal Conjectures statements in Lean.

Lemma selects a formal statement, names the exact Lean target, and posts a reward. A miner submits a Lean proof file. If Lean verifies that file against the selected statement, the bounty can be paid.

Source of targets

Formal Conjectures gives Lemma exact Lean statements to point at.

Google DeepMind Formal Conjectures collects mathematical statements written in Lean 4. Some entries are research solved: the mathematics is known, but a clean Lean proof can still be valuable work.

That makes the bounty easy to understand: Lemma can pick one exact statement, publish it as the target, and reward a proof file that Lean accepts.

Formal Conjectures software is Apache 2.0. Other materials are CC-BY 4.0 or source-specific third-party licenses.

Bounty pages

Each bounty should show the target, reward, and proof status.

The public page can list selected theorem IDs, source links, proof provenance, review status, and bounty terms. Open conjectures can be a later lane; launch bounties should stay focused on exact statements where the proof target is already clear.
