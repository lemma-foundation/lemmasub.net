Formal Conjectures

Bounties for harder Lean targets.

Lemma uses the public Formal Conjectures database from Google DeepMind as the source of bounty targets. A bounty is separate from the timed miner rounds and can be attempted by anyone, not only miners.

Note: Lemma is not affiliated with or partnered with Google DeepMind.

Open bounty

Prove a tiny Lean theorem.

This bounty publishes a complete Lean target and accepts any submission file that proves (2 : Nat) + 2 = 4.

Open bounty

two_plus_two_eq_four

Lean target:

theorem two_plus_two_eq_four : (2 : Nat) + 2 = 4 := by
  sorry

Accepted proof:

import Mathlib

namespace Submission

theorem two_plus_two_eq_four : (2 : Nat) + 2 = 4 := by
  rfl

end Submission

Source of targets

Formal Conjectures gives Lemma exact Lean statements to point at.

Some entries are research solved: the mathematics is known, but a clean Lean proof artifact can still be valuable work.

That makes the bounty simple: pick one harder statement, publish the exact target, and reward a proof file that Lean accepts.
