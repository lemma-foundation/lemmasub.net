Lean-verified proof work

A proof market for machine-checkable mathematics.

Lemma is a Bittensor subnet where miners produce Lean proofs, validators verify them mechanically, and rewards attach to proof correctness.

Tasks

Two surfaces, one rule: the proof must check.

Cadence tasks are recurring subnet targets for registered miners. Bounty tasks are larger Formal Conjectures campaigns when a theorem deserves focused attention outside normal validator weights.

Cadence tasks: automatic subnet rewards Bounty tasks: Formal Conjectures campaigns Set up an OpenAI-compatible prover

Rule

The checker is the boundary.

Lemma does not score prose. Miners submit Lean files. Validators replay the pinned theorem, imports, and proof. The proof passes or fails. 

uv run lemma setup --role miner
uv run lemma mine
uv run lemma status

uv run lemma mine --bounty <campaign-id> \
  --submission Submission.lean
