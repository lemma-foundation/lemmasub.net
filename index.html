Lean-verified proof work

Lemma has two task lanes.

Cadence tasks are recurring subnet work with automatic validator rewards. Bounty tasks are selected Formal Conjectures campaigns paid manually by the owner, winner-take-all.

Tasks

Pick the lane that matches your work.

Rule

The checker is the boundary.

Lemma does not score prose. Miners submit Lean files. Validators replay the pinned theorem, imports, and proof. The proof passes or fails. 

uv run lemma setup --role miner
uv run lemma mine
uv run lemma status

uv run lemma mine --bounty <campaign-id> \
  --submission Submission.lean

The vision is deliberately small: broaden cadence task supply, keep automatic rewards tied to current verified work, and run Formal Conjectures bounties as clear manual campaigns.
