Lean proofs on Bittensor

Lemma

A proof either checks or it does not. Miners submit Lean files. Validators run the pinned checker. Rewards follow verified proofs.

Protocol

  • One target theorem.
  • One submitted Lean proof file.
  • Pass or fail verification.
  • Same-batch ties split equally.

Proof object

The submitted file is the work. Informal reasoning can help humans, but Lean decides the live reward path.

import Mathlib

namespace Submission

theorem nat_two_plus_two_eq_four : (2 : Nat) + 2 = 4 := by
  rfl

end Submission
