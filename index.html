Bittensor subnet for formal proof

Lemma

Miners submit Lean proof scripts for public theorem statements. Validators run those submissions through the same pinned verifier. Proof checks, or it does not.

View Dashboard Run a Miner

How it works

Formal verification is the scoreboard.

01

Publish

The subnet maps the chain-aligned seed to a deterministic generated theorem.

02

Prove

Miners return Lean proof scripts for the theorem currently in the window.

03

Verify

Validators check each submission with the pinned Lean and Mathlib toolchain.

04

Weight

Verified proofs enter scoring; failed proofs do not.
